# Quick Start Guide

## Prerequisites
- Ensure you have Python 3.7 or higher installed.
- Install necessary libraries using the following command:
  ```bash
  pip install -r requirements.txt
  ```

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/asqvisuals/Stable-Video-Infinity.git
   cd Stable-Video-Infinity
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage Examples
- To run the application, use the following command:
  ```bash
  python main.py
  ```
- For specific options, refer to:
  ```bash
  python main.py --help
  ```

## Troubleshooting
- If you encounter issues, ensure that all dependencies are installed correctly.
- Common errors include missing libraries; if this occurs, try reinstalling them.
- If the problem persists, consult the [issues section](https://github.com/asqvisuals/Stable-Video-Infinity/issues) of the GitHub repository.
